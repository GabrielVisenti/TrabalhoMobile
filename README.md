Este projeto consiste no desenvolvimento de uma calculadora mobile inspirada no layout da calculadora do iPhone (iOS), utilizando React Native. A aplicação foi criada com o objetivo de permitir a realização de operações matemáticas básicas de forma simples, rápida e intuitiva.

A calculadora possibilita que o usuário insira números por meio de botões na interface e execute operações como adição, subtração, multiplicação e divisão, exibindo o resultado imediatamente na tela.

O projeto tem como foco principal a aplicação de conceitos fundamentais do desenvolvimento mobile, incluindo:

. Manipulação de estados (useState) em React Native
. Criação e organização de componentes reutilizáveis
. Estruturação de interfaces responsivas utilizando Flexbox
. Implementação de eventos de interação do usuário (onPress)
. Organização e separação da lógica da aplicação

Além disso, o projeto busca reproduzir uma experiência visual semelhante à calculadora do iOS, priorizando simplicidade, usabilidade e organização do código.

Este projeto também tem caráter educacional, sendo desenvolvido como parte do processo de aprendizado em programação para dispositivos móveis, com o objetivo de praticar conceitos de desenvolvimento com React Native e Expo.

Funcionalidades:

Inserção de números
Operação de adição (+)
Operação de subtração (-)
Operação de multiplicação (*)
Operação de divisão(/)
Operação com porcentagem (%)
Botão para apagar o último número
Exibição do resultado da operação

Funções Principais:

Adicionar()
EscolherOperação()
Calcular()
Limpar()
Apagar()

Requisitos Funiconais:

O sistema deve permitir que o usuário insira números através dos botões da calculadora.
O sistema deve permitir realizar a operação de adição (+).
O sistema deve permitir realizar a operação de subtração (-).
O sistema deve permitir realizar a operação de multiplicação (×).
O sistema deve permitir realizar a operação de divisão (÷).
O sistema deve permitir que o usuário visualize os números digitados no visor.
O sistema deve exibir o resultado da operação matemática ao pressionar o botão “=”.
O sistema deve permitir limpar todos os valores da calculadora através do botão “C”.
O sistema deve permitir apagar o último número digitado através do botão “⌫”.
O sistema deve permitir calcular porcentagem (%) de um número digitado.

Requisitos Não Funiconais:

O sistema deve possuir uma interface simples e intuitiva para facilitar o uso.
O sistema deve possuir um layout organizado em formato de calculadora.
O sistema deve possuir botões grandes e visíveis, facilitando a interação do usuário.
O sistema deve possuir um tempo de resposta imediato ao pressionar os botões.
O sistema deve ser desenvolvido utilizando React Native.
O sistema deve ser compatível com dispositivos móveis Android e iOS.
O sistema deve apresentar baixo consumo de memória, devido à simplicidade do aplicativo.
O sistema deve apresentar design inspirado na calculadora do iPhone (iOS).
O código do sistema deve ser organizado e legível para manutenção futura.
O sistema deve permitir execução através do ambiente Expo.
